# Dashboard Screenshot Guide

Add portfolio screenshots here after running the Streamlit demo or building the Power BI report.

Recommended files:

- `executive_overview.png`
- `segment_analysis.png`
- `risk_evaluation.png`
- `customer_drilldown.png`

Suggested capture flow:

1. Run `python src/segment_customers.py`.
2. Run `streamlit run src/app.py`.
3. Capture the Executive Overview, Segment Analysis, Risk Evaluation, and Customer Drilldown pages.
4. Use the same country and industry filters across screenshots so the story feels consistent.
